HTML

  • The head element contains information about the webpage.
  • The body elements represent the visible content shown to the user.
  • There are parent and child elements
  • Heading elements should be in order for use with screen readers
  • Declarations are not case-sensitive; HTML elements are case-sensitive.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Best practice is to use an external css sheet and link to it in the head of the html
  • Colors can be incorpated either by common name or by hex #.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new breanch and switches to it
  • cd = open
  • git add A- = add changes
  • git commit -m "description" = comment on commited changes
  • git pull origin main = pulls main branch
  • git push origin "name of branch"
  • code . = open VS

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function must be called before it will execute
  • Access Chrome Dev tools by right clicking and hitting "Inspect."
  • Can apply if statement or else if statements
  • Strings are a sequence of text. Strings are enclosed in single or double quote marks.
  • Boolean is a True/False value. The words true and false are special keywords that don't need quote marks.
  • Numbers also do not need quotation marks
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • Numbers correspond to the variables in arrays and start with zero as opposed to 1.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
